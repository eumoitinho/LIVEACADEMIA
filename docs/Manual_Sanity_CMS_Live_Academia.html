1 Introdução

Bem-vindo ao manual completo do Sanity CMS da Live Academia. Este guia foi desenvolvido para capacitá-lo a gerenciar todo o conteúdo do site de forma autônoma e profissional.

🎯 O que é o Sanity CMS?

O Sanity CMS é o sistema de gerenciamento de conteúdo que permite você editar todo o conteúdo do seu site sem precisar mexer em código. É como ter um painel de controle completo da Live Academia na palma da sua mão.

💪 O que você pode fazer?

🏠
Editar Homepage
Altere títulos, descrições, CTAs e imagens da página principal
🏢
Gerenciar Unidades
Adicione novas unidades, edite informações e serviços
💳
Planos e Preços
Atualize preços, benefícios e informações dos planos
🎯
Aulas Coletivas
Gerencie modalidades, horários e instrutores
✨
Benefícios
Edite e adicione novos benefícios da academia
💬
Depoimentos
Adicione novos depoimentos de alunos satisfeitos

📚 Conceitos Importantes

Termo Significado
Documento Cada página, unidade ou plano é um "documento" no Sanity
Campo Cada informação editável (título, descrição, preço, etc)
Publicar Suas alterações só aparecem no site após clicar em "Publish"
Rascunho Você pode salvar alterações sem publicar

2 Acessando o Sanity Studio

Aprenda como fazer login e acessar o painel de controle do Sanity CMS.

🌐 Como Acessar

Para acessar o Sanity Studio, você pode usar uma das URLs abaixo:

https://liveacademia.com.br/studio
ou
http://localhost:3000/studio (para desenvolvimento local)
💡 Dica

Salve a URL do Studio nos favoritos do seu navegador para acesso rápido!

🔐 Fazendo Login

  1. Acesse a URL do Sanity Studio
  2. Faça login com suas credenciais Google ou GitHub
  3. Aguarde o carregamento completo do painel
  4. Você será direcionado ao dashboard principal
⚠️ Importante

Mantenha suas credenciais de acesso em segurança. Não compartilhe com pessoas não autorizadas.

4 Editando a Homepage

Aprenda a editar cada seção da página principal da Live Academia.

🏠 Acessando a Homepage

  1. No menu lateral, clique em "Homepage"
  2. Você verá o documento "Homepage Principal"
  3. Clique no documento para abrir o editor

🎯 Hero Section (Seção Principal)

Esta é a primeira seção que os visitantes veem ao entrar no site.

Campos Disponíveis:

  • Title: Título principal (ex: "Transforme.")
  • Subtitle: Subtítulo (ex: "Evolua.")
  • Third Title: Terceiro título (ex: "Viva.")
  • Description: Parágrafo explicativo sobre a academia
  • Rating Value: Nota da avaliação (ex: "4.9")
  • Rating Label: Label da avaliação (ex: "Elite rating")
  • Rating Subscribers: Número de alunos (ex: "15k+ atletas")
  • Primary CTA Text: Texto do botão principal (ex: "Comece Agora")
  • Primary CTA Link: Link do botão (ex: "/planos")
  • Secondary CTA Text: Texto do botão secundário
  • Secondary CTA Link: Link do botão secundário
  • Footer Text: Texto do rodapé da seção
✏️ Como Editar
  1. Localize o campo que deseja alterar
  2. Clique no campo
  3. Digite o novo texto
  4. Clique em "Publish" no topo da página

📖 About Section (Seção Sobre)

Seção que conta a história e valores da Live Academia.

Editando a Seção Sobre:

  1. Role até a seção "About"
  2. Edite os campos:
    • Badge (ex: "Sobre a Live Academia")
    • Title (ex: "Seu treino, suas regras")
    • Description (descrição da academia)
    • Stats (estatísticas como anos de experiência)
    • Highlights (lista de destaques)
  3. Clique em "Publish"

✨ Benefícios Section

Grid com os principais benefícios da academia.

Adicionando um Novo Benefício:

  1. Role até "Benefícios"
  2. Clique em "+ Add item"
  3. Preencha os campos:
    • Icon (nome do ícone)
    • Title (título do benefício)
    • Description (descrição)
    • Color (cor do gradiente)
    • Image (URL da imagem)
  4. Clique em "Publish"

🎛️ Section Controls

Controla quais seções aparecem ou ficam ocultas no site:

  • Mostrar seção sobre
  • Mostrar seção benefícios
  • Mostrar seção planos
  • Mostrar seção depoimentos

✅ Marcado = Seção aparece no site
❌ Desmarcado = Seção fica oculta

🔍 SEO

Otimização para mecanismos de busca (Google, Bing, etc).

Campo Descrição Limite

Meta Description Descrição que aparece no Google 160 caracteres

10 Boas Práticas

Dicas e recomendações para usar o Sanity CMS de forma eficiente.

✅ Checklist Antes de Publicar

  • Revisei todos os textos (ortografia e gramática)
  • Testei todos os links (CTAs, botões, links externos)
  • Verifiquei se as imagens carregam corretamente
  • Preenchi os campos de SEO (title, description, keywords)
  • Testei em diferentes dispositivos (mobile, tablet, desktop)
  • Verifiquei preços e informações de contato
  • Confirmei horários e datas

✍️ Dicas de Escrita

Títulos:

  • Use verbos de ação (Transforme, Descubra, Aprenda)
  • Seja específico e claro
  • Use números quando possível (3 Unidades, 24h)
  • Mantenha o tom motivacional

CTAs (Chamadas para Ação):

  • Seja específico: "MATRICULE-SE AGORA" > "Clique aqui"
  • Crie urgência: "ÚLTIMAS VAGAS", "OFERTA LIMITADA"
  • Use verbos no imperativo: TRANSFORME, DESCUBRA
  • Mantenha consistência em todo o site

📸 Dicas de Imagens

Tipo Tamanho Recomendado Formato
Hero Section 1920x1080px JPG/WebP
Cards de Unidade 800x600px JPG/WebP
Imagens de Benefícios 600x400px JPG/WebP
Avatar de Depoimentos 400x400px JPG/PNG
⚡ Performance

Comprima imagens antes de usar (máx. 500KB). Use ferramentas como TinyPNG ou Squoosh para otimizar.

🔗 Dicas de Links

  • Links internos começam com / (ex: /planos)
  • Links externos começam com https://
  • Para âncoras, use #nome-da-secao
  • Teste todos os links após publicar
  • Use links descritivos e claros

📱 Responsividade

  • Teste em dispositivos móveis
  • Verifique se textos não ficam muito pequenos
  • Confirme se botões são fáceis de tocar
  • Teste em diferentes navegadores
  • Verifique a velocidade de carregamento

11 Suporte

Precisa de ajuda? Aqui estão os recursos de suporte disponíveis.

🔧 Problemas Comuns e Soluções

1. Não consigo publicar alterações

  • Verifique se você está logado
  • Certifique-se de clicar em "Publish" (não apenas "Save")
  • Aguarde alguns segundos e recarregue o site
  • Limpe o cache do navegador (Ctrl+Shift+Delete)

2. Imagem não aparece no site

  • Verifique se a URL está correta
  • Teste abrindo a URL em uma nova aba
  • Use URLs completas: https://... ou /images/...
  • Verifique se a imagem não está muito pesada

3. Link não funciona

  • Links internos devem começar com /
  • Links externos devem começar com https://
  • Para âncoras, use #nome-da-secao
  • Teste o link em uma nova aba

4. Preço não aparece corretamente

  • Lembre-se: preço deve estar em centavos
  • R$ 119,90 = 11990 (sem pontos ou vírgulas)
  • Verifique se o campo está preenchido
